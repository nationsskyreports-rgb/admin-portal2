02 / 07

الأمان والصلاحيات

من المهم فهم الفرق الكبير بين صلاحيات الأدمن والموظف.

⚠️ تحذير أمني

مفتاح service_role يمنح القدرة على قراءة وكتابة أي بيانات في Supabase.

  • يجب ألا يكون موجوداً في كود الموظفين.
  • مستودع GitHub يجب أن يكون Private.

🛡️ جدول الإداريين

الدخول يتم عبر admins table.

  • البيانات المخزنة: username, password_hash, role.
  • لإضافة أدمن جديد: إضافة صف مباشرة في Supabase.
  • تغيير كلمة المرور: تعديل عمود password_hash في Supabase مباشرة.
ملحوظة: الـ Admin Portal يستخدم نظام مصادقة منفصل (Custom)، بينما الـ Agent Portal يستخدم Supabase Auth.

إدارة الجدول (Schedule & Breaks)

الصفحة الأهم في النظام، حيث يتم تحديد عمل الموظفين.

تدفق نشر الجدول (Publish Flow)

بناء الجدول (Create Week) → نشر الجدول (Publish)
توليد البريكات تلقائياً (Auto-generate Breaks)
تحديث الإجازات السنوية (Auto-update Annual Leave)
ظهور الجدول فوراً للموظفين
⏰ Breaks Page
تعديل أوقات البريكات يدوياً.
عرض "Queue Timeline" للتأكد من تغطية الخط.
ميزة نسخ أوقات الأمس (Copy from Yesterday).
🔄 Requests
الموافقة أو الرفض على طلبات الموظفين (إجازات، تبديل وردية).
الموافقة هنا فقط تغير الحالة، التعديل الفعلي يتم يدوياً.
04 / 07

استيراد البيانات والجودة

📥 استيراد xCally

طريقة تحديث مؤشرات الأداء (KPIs) شهرياً.

  1. تصدير ملف CSV من نظام xCally.
  2. رفع الملف في صفحة xcally-import.html.
  3. النظام يطابق الأسماء عبر حقل xcally_name.
  4. يحفظ في جدول daily_performance.
⚠️ مهم: يجب أن يتطابق اسم الموظف في الجدول تماماً مع الاسم في CSV.

⭐ تقييم الجودة (Quality)

تسجيل تقييمات المكالمات وحساب النسبة المئوية.

  • ✅ إضافة مراجعات (Reviews) لكل موظف.
  • ✅ النظام يحسب المتوسط التلقائي للشهر.
  • ✅ النسبة النهائية تظهر في quality_scores.
05 / 07

دليل الصيانة والتحديث

الخطوات الروتينية للحفاظ على سير العمل.

🔄 تحديث النظام
تعديل الملف محلياً → رفعه على GitHub (admin-portal2-main) → النشر التلقائي خلال دقيقتين.
➕ إضافة موظف جديد
1. الإضافة من صفحة Agents.
2. إنشاء مستخدم Supabase Auth يدوياً.
3. إضافته في جدول الشهر الجاري.
🔑 إعادة تعيين كلمة مرور الأدمن
الذهاب إلى Supabase Dashboard → جدول admins → تعديل عمود password_hash (كلمة مرور نصية).
